# Kotlin Full Groove Room

Полная Kotlin/Ktor-версия сайта The Groove Room.

Внутри проекта нет отдельных файлов `index.html`, `style.css` и `script.js`.
Страница, стили и карточки товаров создаются Kotlin-кодом в `Application.kt`.

## Запуск

```bash
gradle run
```

Потом открыть:

```text
http://localhost:8080
```

## Где код

```text
src/main/kotlin/com/nikitateplov/groove/Application.kt
```
